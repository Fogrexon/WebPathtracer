# PathTracer

## Build

```
npm install

// ts
npm run build

// wasm
npm run build:wasm

// docs
npm run build:docs
```

## Develop

```
// ts with file watching
npm start

// docs with file watching
npm docs
```

## Use

## Example

```
npm run start:example
```
